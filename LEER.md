# MapReduce

En CouchDB  se creo una base de datos con 16 canciones (canciones.json) de una playlist de Spotify.

Dentro de nuestra BD se diseñaron las siguientes vistas (funciones.java):

 -Minutos de artista
 -Minutos por album
 -Por album
 -Por artista
 -Por año de lanzamiento
 -Por duracion
 -Por genero
 -Por productor
 -Sencillos
 -Ultimo año
